# 🔐 Digital Security Toolkit

A simple Python-based CLI application that helps users stay safe online by providing tools for phishing detection, password checking, quizzes, and security tips.

---

## 🚀 Features

- 📩 Phishing Detection
- 🔐 Password Strength Checker
- 🧠 Cybersecurity Quiz
- 💡 Security Tips

---

## 🛠️ Tech Stack

- Python
- Modular Programming

## ▶️ How to Run

1. Clone the repository  
2. Open terminal
